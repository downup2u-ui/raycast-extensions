# Webhook Failure Verdict

A Raycast extension that provides **decision-first verdicts** for common webhook failure scenarios.

This extension helps developers quickly decide **whether a webhook issue is worth further debugging**, before spending time on logs, retries, or infrastructure changes.

## Included Verdicts

- Webhook Not Triggered  
- Webhook Delivered But Ignored  

Each verdict:
- States a clear STOP / boundary
- Defines responsibility ownership
- Links to a full reference page (optional)

## Privacy

This extension:
- Does not collect user data
- Does not track usage
- Does not send information to external services
- Only opens external links on explicit user action
